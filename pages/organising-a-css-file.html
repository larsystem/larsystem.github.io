organising a css file

Some basic tips to ensure your css doesn't get out of hand!

When dealing with CSS, there are a lot of classes that you'll need to make for any well-formatted or enticing website.

1. Indent everything.

It's very important to do this from the moment you start because it becomes more monotonous and time-consuming the longer you wait.

2. Use comments.

Without comments, you'll be driven mad by the task of repeatedly finding out what each thing does. To write a comment in css, use a /* to open, and a */ to close.
